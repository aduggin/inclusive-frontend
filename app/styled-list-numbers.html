Challenge

To use css to style the numbers rather than using images as this will be more performant (fewer http requests) and more scalable (easy to change the colors, not dependent on having the image for a specific number)

To make sure that browser that do not support the css still present the content as a number list

Acceptance criteria

  • Modern browsers display styled numbers
  • Browser that do not support border radius and ::before render a normal ordered list
  • Screen reader read out the list like a normal ordered list - with no duplication of numbers
